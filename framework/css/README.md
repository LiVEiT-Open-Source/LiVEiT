# LiVEiT® CSS Reference

This folder contains the official styling for the LiVEiT® Framework.

The CSS layer is responsible for the visual appearance of a LiVEiT® implementation while remaining independent from the HTML structure and JavaScript behaviour.

## Purpose

The CSS reference provides:

- Circle styling
- Colour definitions
- Responsive layouts
- Typography
- Glass effects
- Hover effects
- Animation
- Accessibility improvements

## Design Philosophy

The official LiVEiT® design system aims to be:

- Clean
- Modern
- Accessible
- Responsive
- Framework independent

Developers are encouraged to customise the appearance while maintaining accessibility and usability.

The framework defines the structure.

CSS defines the visual language.
