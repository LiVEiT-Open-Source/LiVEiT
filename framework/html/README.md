# LiVEiT® HTML Reference

This folder contains the official HTML reference structure for the LiVEiT® Framework.

The HTML layer defines the structure of a LiVEiT® implementation.

It does not define visual styling or behaviour.

## Purpose

The HTML reference is intended to provide:

- A standard page structure
- A reusable circle layout
- Modal structure
- Content sections
- Accessibility friendly markup
