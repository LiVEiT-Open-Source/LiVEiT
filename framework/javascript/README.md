# LiVEiT® JavaScript Reference

This folder contains the official JavaScript reference for the LiVEiT® Framework.

The JavaScript layer is responsible for behaviour and interaction.

It should remain separate from HTML structure and CSS presentation.

## Purpose

The JavaScript reference provides:

- Modal behaviour
- Circle interaction
- Navigation controls
- Keyboard support
- Focus handling
- Accessibility improvements
- Optional animation behaviour
