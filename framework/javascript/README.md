# LiVEiT® JavaScript Reference

This folder contains the official JavaScript reference for the LiVEiT® Framework.

The JavaScript layer is responsible for behaviour and interaction.

It should remain separate from HTML structure and CSS presentation.

## Purpose

The JavaScript reference provides:

- Modal behaviour
- Circle interaction
- Navigation controls
- Keyboard support
- Focus handling
- Accessibility improvements
- Optional animation behaviour

## Design Philosophy

JavaScript should enhance the framework without making it unnecessarily complex.

The official LiVEiT® implementation should remain:

- Lightweight
- Predictable
- Accessible
- Easy to maintain
- Framework independent

## Progressive Enhancement

Where possible, LiVEiT® implementations should remain understandable even if JavaScript is unavailable.

HTML provides the structure.

CSS provides the visual language.

JavaScript provides behaviour.

The framework defines the structure.

The implementation defines the meaning.
