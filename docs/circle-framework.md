# The LiVEiT® Circle Framework

## Introduction

The LiVEiT® Circle Framework is the core of the LiVEiT® methodology.

Rather than presenting information as a traditional linear process, the framework uses a circular structure to encourage exploration, reflection and understanding from multiple perspectives.

The framework has been designed to support communication, self-understanding and collaboration while remaining flexible enough to be adapted for healthcare, education, charities, communities and many other fields.

At its heart, the LiVEiT® Framework separates **structure** from **implementation**.

The framework provides a consistent visual structure.

Each implementation defines how that structure is used within its own domain.

---

# Why a Circle?

Life rarely follows a straight line.

People naturally revisit ideas, experiences and emotions throughout their lives.

The circular design reflects this process.

Rather than encouraging people to move from beginning to end, the LiVEiT® Circle encourages continuous exploration, allowing individuals to approach subjects from different directions and revisit areas whenever appropriate.

There is no single "correct" starting point.

The framework adapts to the individual.

---

# The Twelve Positions

The LiVEiT® Framework consists of twelve positions arranged around a circle.

Each position represents a fundamental principle of understanding rather than a fixed subject.

This allows the framework to be adapted across many different disciplines while maintaining a consistent visual language.

The table below describes the standard LiVEiT® Framework.

| Position | Colour | Framework Principle | Healthcare Example |
|----------|---------|---------------------|--------------------|
| 1 | Blue | Emotional Awareness | Anxiety |
| 2 | Orange | Physical Wellbeing | Inflammation & Diet |
| 3 | Green | Biological Systems | Endocannabinoid System |
| 4 | Brown | Physical Comfort | Pain Management |
| 5 | Grey | Care & Support | Palliative Care |
| 6 | White | Professional Guidance | Consultation |
| 7 | Red | Psychological Wellbeing | Mental Health |
| 8 | Black | Trauma & Recovery | PTSD |
| 9 | Yellow | Creativity & Expression | Artistic Wellbeing |
| 10 | Violet | Personal Safety | Self Harm Support |
| 11 | Rose | Community & Compassion | Charities |
| 12 | Aqua | Trust & Security | Private Healthcare |

These principles are provided as the standard LiVEiT® implementation. Organisations are encouraged to adapt them to suit their own requirements while preserving the integrity of the framework.

---

# Framework vs Implementation

One of the core principles of LiVEiT® is the separation of the framework from its implementation.

The framework defines the structure.

Implementations define the meaning.

For example:

Healthcare organisations may use Position 1 to represent Anxiety.

Schools may use Position 1 to represent Emotional Development.

Businesses may use Position 1 to represent Staff Wellbeing.

Research projects may use Position 1 to represent Participant Reflection.

The position remains the same.

Only the implementation changes.

---

# Adaptability

One of the strengths of the LiVEiT® Framework is its flexibility.

The same circular structure can be adapted for many different purposes.

The same framework can support completely different implementations without changing its architecture.

Only the interpretation of each position changes.

This consistency allows users to become familiar with the framework while enabling organisations to tailor it to their own needs.

Examples include:

- Healthcare
- Education
- Mental Health
- Neurodiversity
- Community Projects
- Charity Services
- Project Management
- Personal Development
- Research
- Training

The framework provides the structure.

Each implementation provides its own content.

---

# Design Principles

The LiVEiT® Framework is guided by several key principles.

- Simplicity
- Accessibility
- Inclusivity
- Collaboration
- Reflection
- Flexibility
- Visual Understanding
- Open Knowledge

These principles should remain at the centre of every implementation.

---

# Open Framework

The LiVEiT® Circle Framework is released as open source.

Developers are encouraged to:

- Build upon the framework.
- Improve the framework.
- Share improvements with the community.
- Create new educational examples.
- Develop new implementations.

---

# Official Applications

The official LiVEiT® applications demonstrate one implementation of the framework.

They are not the framework itself.

The framework is intentionally broader than any individual application and is designed to support many different implementations.

---

# Looking Forward

The LiVEiT® Circle Framework will continue to evolve through community collaboration.

As new ideas emerge, the framework will grow while remaining true to its core purpose:

**Helping people better understand themselves and one another through open technology and shared understanding.**

The LiVEiT® Framework is not intended to define how organisations should work.

Instead, it provides a shared structure that encourages understanding, collaboration and continuous improvement across many different fields.

The framework defines the structure.

The implementation defines the meaning.
