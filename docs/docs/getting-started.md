# Getting Started with the LiVEiT® Framework

## Welcome

Welcome to the LiVEiT® Framework.

Whether you're a developer, designer, educator, healthcare professional, researcher or simply curious about the framework, we're pleased you've decided to explore the project.

# What Is the LiVEiT® Framework?

The LiVEiT® Framework is an open source visual framework.

It provides a consistent circular structure that can be adapted to support many different implementations.

Examples include:

- Healthcare
- Education
- Community Projects
- Charities
- Neurodiversity
- Personal Development
- Research
- Business
- Training

The framework remains consistent.

The implementation defines its purpose.

---

# What Is Included?

The open source framework includes:

---

# What Is Not Included?

This repository does not contain the proprietary source code for the official LiVEiT® applications.

The official applications demonstrate one implementation of the framework and are developed independently.

---

# Your First Goal

Your first objective is not to build an application.

Your first objective is to understand the framework.

We recommend reading the Circle Framework guide before writing any code.

---

# Building Your First Implementation

Once you understand the framework you can begin creating your own implementation.

Every implementation should:

- Respect the framework principles.
- Preserve accessibility.
- Encourage understanding.
- Be visually clear.
- Be easy to navigate.
- Remain adaptable.

The meaning of each position may change.

The structure should remain consistent.
