# LiVEiT® Branding Guide

Welcome to the official **LiVEiT® Branding Guide**.

This guide defines how the LiVEiT® identity should be represented across digital, print and commercial media.

The purpose of this documentation is to ensure that the LiVEiT® brand remains consistent, recognisable and accessible while allowing developers, designers and organisations to build upon the LiVEiT® Framework.

---

# Brand Philosophy

LiVEiT® was created around a simple principle:

> **Technology should simplify understanding, not create complexity.**

The LiVEiT® Framework uses visual structure to organise information into clearly defined areas, allowing users to navigate projects, concepts and workflows through an intuitive circular interface.

Every element of the visual identity supports this philosophy.

The brand should always communicate:

* Clarity
* Structure
* Accessibility
* Trust
* Simplicity
* Professionalism
* Human-centred design

LiVEiT® avoids unnecessary decoration in favour of clean layouts, generous spacing and consistent visual language.

---

# Design Principles

Every official LiVEiT® project should aim to follow these principles.

## Simple

Interfaces should remain clean and free from unnecessary visual clutter.

## Consistent

Colours, typography, spacing and component behaviour should remain predictable throughout every project.

## Accessible

Design decisions should support users of varying abilities through sufficient contrast, readable typography and keyboard-friendly interaction.

## Recognisable

Official branding should be immediately identifiable regardless of platform.

## Scalable

The visual identity should work equally well across:

* Websites
* Mobile applications
* Desktop software
* Documentation
* Presentations
* Printed material
* Large-format displays

---

# Open Source Philosophy

The LiVEiT® Framework is published as an open-source project to encourage learning, collaboration and community contribution.

Developers are encouraged to:

* Learn from the framework
* Build upon the framework
* Submit improvements
* Create demonstrations
* Develop educational resources

Open source collaboration strengthens the framework while maintaining a consistent public identity.

---

# Brand Integrity

Although the framework is open source, the **LiVEiT®** name, logos and visual identity are protected intellectual property.

Official branding exists to identify authentic LiVEiT® projects and should not be altered in ways that create confusion or imply endorsement where none exists.

Guidance for trademark usage and commercial licensing is provided later in this documentation.
