# LiVEiT® Logo Studio

The LiVEiT® Logo Studio is the official standalone branding utility for the LiVEiT® Framework.

It has been designed to generate consistent, high-quality branding assets directly from the official LiVEiT® logo without requiring external graphics software.

Unlike traditional design workflows, the Logo Studio works entirely within a single HTML file and has **no external dependencies**.

---

# Purpose

The Logo Studio provides a fast and consistent method of producing official LiVEiT® branding assets.

It is intended for:

- Official logo generation
- Brand asset creation
- Website graphics
- Application icons
- Marketing material
- Social media graphics
- Presentation assets
- Internal design work

---

# Features

Current features include:

- Standalone HTML application
- No installation required
- No external CSS
- No JavaScript libraries
- No assets folder
- SVG-based rendering
- Individual circle controls
- Show / hide circles
- Background colour selector
- Background image preview
- Hidden text colour controls
- PNG export
- JPEG export
- Apple App Icon export presets
- Multiple export resolutions
- High-resolution output
- Glass-style control interface

---

# Usage

1. Open **logo_studio.html** in any modern web browser.

2. Configure the logo using the built-in controls.

3. Select the required export format.

4. Select the required canvas size.

5. Export the finished image.

No additional software is required.

---

# Export Options

The Logo Studio supports exporting images in multiple formats and resolutions.

Current export formats include:

- PNG
- JPEG

The Apple export options provide commonly used application icon sizes for Apple platforms.
