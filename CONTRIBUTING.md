# Contributing to LiVEiT®

## Welcome

Thank you for your interest in contributing to the LiVEiT® Framework.

Whether you write code, improve documentation, design graphics, suggest new ideas or simply report an issue, every contribution helps strengthen the project.

LiVEiT® exists because different perspectives matter.

Every contribution is another perspective.

---

# Our Philosophy

LiVEiT® was created to encourage collaboration rather than competition.

We believe that open technology, shared knowledge and respectful discussion produce better outcomes than working in isolation.

If you contribute to LiVEiT®, you become part of a community that values understanding, accessibility and continuous learning.

---

## Framework Development

## Documentation

## Accessibility

We particularly welcome improvements that make the framework easier to use for:

* Neurodivergent individuals
* Disabled users
* Screen reader users
* Keyboard navigation
* Colour accessibility

---

## Education

Help us create:

* Learning resources
* Educational examples
* Workshops
* Demonstration projects

---

## Ideas

Not every contribution needs to be code.

If you have an idea that could improve the LiVEiT® Framework, we'd love to hear it.

Please open a GitHub Discussion or Issue describing your suggestion.

---

# Before You Contribute

Please check existing Issues and Discussions before starting work.

If you're planning a large change, consider discussing it with the community first.

---

# Pull Requests

Before submitting a Pull Request, please ensure that:

* Your contribution is relevant to the LiVEiT® Framework.
* Your code is well documented.
* Existing functionality is not broken.
* New features are explained clearly.
* Documentation has been updated where appropriate.

---

# What This Repository Contains

This repository focuses on the open source LiVEiT® Framework.

It includes:

* Framework documentation
* HTML
* CSS
* JavaScript
* SVG assets
* Templates
* Educational resources

The official LiVEiT® applications are developed separately and are not part of this repository.

---

# Be Respectful

LiVEiT® welcomes contributors from all backgrounds, experiences and disciplines.

We ask everyone to:

* Be respectful.
* Be constructive.
* Assume positive intent.
* Welcome different perspectives.
* Encourage learning.
* Help others gr0w.
